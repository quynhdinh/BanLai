# BanLai
## Pre-requisites
1. Install Node JS
2. Install Mini App DevTools CLI:
  
    `npm install -g zmp-cli`
  
3. Download or clone this repository
## Set up
### Client
1. Move to your client subfolder:

    `cd client`
    
2. Start client using zmp-cli

    `zmp start`
    
3. Open localhost:3000
